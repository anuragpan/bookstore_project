python manage.py startapp <polls>
---startapp polls will create dir polls with various fills needed for it.
---then create urls.py in polls folder (i.e. /polls/urls.py)
---create folder templates inside polls again (i.e. /polls/templates/polls/) where our html will land.

python manage.py runserver

python manage.py migrate
---after update the settings.py.

---update the models.py in polls folder.

python manage.py migrate <polls>
